Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом доcтаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2em, при промощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2em, при промощи свойства margin-bottom. Картина вставлена при помощи тега, плавление организовано по правому краю внутри даннойй статьи для этой картинки.
Нижний блок, подвал. Занимает всю доступную ширину. Высота опредлеяется помещённым внутрь контентом.
